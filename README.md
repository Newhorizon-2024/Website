🌐 New Horizon — GitHub Pages 网站
用于展示 New Horizon 项目的多功能网页，包括音乐播放、导航切换、新闻展示、设置面板等模块。本仓库用于托管网站的静态资源，并通过 GitHub Pages 提供在线访问。

📁 项目结构
/icons/ — 图标
/images/ — 图片、图标、音频等素材
/media/ — 音乐文件与歌单配置
/scripts/ — JavaScript 功能逻辑
/styles/ — CSS 样式文件
/index.html — 网站主页面

✨ 网站功能
导航栏切换不同内容板块
音乐播放器
Tooltip 信息提示
响应式布局与视觉效果
